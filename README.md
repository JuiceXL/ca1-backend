# CA1 Backend

## Project Description
This project is a RESTful API built using **Node.js**, **Express**, and **MySQL**.

The API allows users to:
- Create users
- View users
- Create challenges
- Update challenges (owner only)
- Complete challenges
- View challenge attempts
- Delete challenges (owner only)

## Technologies Used
- Node.js
- Express.js
- MySQL
- Postman (for API testing)
